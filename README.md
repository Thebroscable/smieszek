## Instalacja

1. Sklonuj repozytorium na serwer:

```bash
git clone https://github.com/Thebroscable/smieszek
cd smieszek
```

2. Budowanie obrazu i uruchomienie kontenera:

```bash
docker-compose up -d --build
```
